# Implemento de atividade do Senai.

## O problema de sofia.

### Persona 7: Sofia (Universitários e a "Fome Oculta")

**Perfil:** 19 anos, morando sozinha pela primeira vez em uma república.

**O Desafio:** Ela só sabe cozinhar macarrão instantâneo, seu orçamento é mínimo e ela está começando a se sentir constantemente cansada por falta de nutrientes, mas acha que cozinhar é perda de tempo e muito caro.

## Solução encontrada

A criação de um site com receitas fáceis, baratas e nutritivas para ajudar pessoas como Sofia a terem uma alimentação mais saudável sem gastar muito dinheiro e sem perder muito tempo.

## Implementação

- login e senha
- A criação de uma api para gerenciar as receitas
- Visualização e modo de preparos
- Filtro de ingredientes
- Estimação de preço
- O frontend será feito com html/css
- O backend será feito com Node.js
